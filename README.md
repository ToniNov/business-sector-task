# "Приложение таблица с данными пользователей и возможностью сортировки и поиска"

## Installation
```bash
npm install
```
#### or
```bash
yarn install
```
#### or
```bash
pnpm install
```
## Running the app

```bash
npm run dev
```
#### or
```bash
yarn dev
```
#### or
```bash
pnpm dev
```
